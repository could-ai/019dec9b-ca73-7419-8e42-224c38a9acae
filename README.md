# SMM Boosting Global App

A modern, cross-platform Social Media Marketing (SMM) platform designed to replicate the core experience of smmboostingglobal.com. This mobile application lets content creators, businesses, and influencers easily purchase and track social media growth services for platforms like Instagram, TikTok, YouTube, and more.

## Features

- **Service Catalog:** Browse a categorized list of social media boosting services by platform.
- **Service Details:** View packages, pricing, delivery speeds, and platform guarantees.
- **Order Tracking Dashboard:** Monitor order statuses (Pending, In Progress, Completed) via an interactive interface.
- **User Profile:** Manage your account, check balances, and easily contact the 24/7 support team.
- **Beautiful UI:** A polished, modern interface with a responsive layout.

## Tech Stack

- **Flutter:** Cross-platform framework for iOS, Android, and Web.
- **Dart:** Programming language for the Flutter framework.

## Setup Instructions

1. Ensure you have the Flutter SDK installed on your system.
2. Clone this repository to your local machine.
3. Run `flutter pub get` in the root directory to install required dependencies.
4. Run the app on your preferred emulator or physical device using `flutter run`.
